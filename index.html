My Work

Magma

This project is a modern web page showcasing real estate technology through digital twin solutions. The site opens with a visually captivating video background and the headline "Experience Real Estate Agility," highlighting the innovative concept. The design is minimalist, with smooth transitions and a clear call to action. Weather

A dynamic and responsive weather application built with HTML, CSS, and JavaScript, offering real-time weather updates through API integration. Key features include a sleek, user-friendly interface, search functionality for any location, and detailed weather data such as temperature, wind speed, and humidity. The app is optimized for performance across devices, with a clean design and intuitive visuals. This project highlights my expertise in front-end development, API handling, and crafting seamless user experiences.
